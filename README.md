# Customer Shopping Behavior Analysis

## Overview

Analyzed retail customer shopping data to identify purchasing trends, customer segments, and product performance to support business decision-making.

## Dataset

* 3,900 transactions | 18 features
* Demographics, purchase details, discounts, reviews, subscriptions

## Tools Used

* **Python**: Data cleaning & EDA
* **SQL (MS SQL Server)**: Business analysis
* **Power BI**: Interactive dashboard

## Key Insights

* Revenue patterns by gender, age group, and subscription status
* Impact of discounts and shipping type on spending
* Identification of loyal and repeat customers

## Outcome

Provided actionable insights to improve marketing strategies, subscriptions, and customer retention.
